Music Box Project

Using Xilinx Spartan-3E and Arduino AT-Mega 2560


The Xilinx module is an FPGA module (Field Programmable Gate Array), FPGA is a digital IC that is often used for digital circuit implementations. Xilinx has 3 parts, namely configure Logic Blocks, I / O Blocks, and Block RAMs. The duration of the project is only 1 week, starting from the search for data note beams which will later be input from data to the installation of mini speakers as the output of this project. This project aims to create music tools with fast processing and become one of the requirements of one of the courses.
